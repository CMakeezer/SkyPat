New Features

In this release, there are several interesting feature added, predicate assertion macros, ability to integrate with the other projects, building system. First, "pat" has added predicate assertion macros, ASSERT_EQ/NE/LT/LE/GT/GE, to have ability to observe the value of two parameters. For example, ASSERT_LT(foo(), bar()) means that foo() should be less than bar(). If the assert fails, "pat" will show the value of foo() and bar() to let programmer know more details about this fail. Second, "pat" has enhanced the ability to integrate with the other projects via providing m4 scripts. User can use these scripts to integrate "pat" into their project as test and performance analysis framework. Finally, "pat" has added a building script to help user build "pat" by autotool more easier. User can execute the building script to build "pat" without setting configure file. Enjoy.
